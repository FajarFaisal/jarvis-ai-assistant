# Jarvis AI Assistant 🤖

## Overview
Jarvis is a personal AI assistant designed to automate tasks, respond to voice commands, and assist with daily productivity.

## Features
- Voice command recognition
- Task automation
- Web search integration
- System control (if applicable)
- Conversational responses

## Tech Stack
- Python
- Speech Recognition / Text-to-Speech libraries
- OpenAI API
- Anthroptic API
- OS automation libraries

## How to Run
1. Clone repo
2. Install dependencies
3. Run main through bash
